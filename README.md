- Dynamic content rendering
- Database-driven pages
- Clean and responsive UI

## Tech Stack
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Project Structure
- Frontend built using HTML, CSS, and JavaScript
- Backend logic handled with PHP
- MySQL database for storing destination data

## Setup Instructions
1. Clone the repository
2. Move the project folder to `htdocs` (XAMPP) or `www` (WAMP)
3. Import the database file into MySQL
4. Update database credentials if required
5. Run the project on `localhost`
